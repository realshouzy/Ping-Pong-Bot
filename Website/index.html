Ping Pong Bots Commands

If you want all commands, here is the list:

  • pp!help
    • This shows all the commands and a some explanation about the bot and its propose
  • pp!ping
    • This shows you the bots latency
  • pp!vote
    • This shows you the link to vote the for the bot on Top.gg if you like and want to support it
  • pp!invite
    • This shows you the invite link of the bot
  • pp!code
    • This shows you the link to the GitHub page of the bot
  • pp!stats
    • This shows you the number of servers the bot is in
  • pp!website
    • This shows you the link to this website
